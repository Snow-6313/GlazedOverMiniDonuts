Recommended auth flow
  1. POST username/password to the login endpoint over HTTPS.
  2. Backend checks hashed password and returns a secure session token.
  3. Frontend stores the token and sends it with later save requests.
  4. Use the users endpoint later for admin creation, resets, and roles.
